# 🌟 Blackline Tattoo Studio

O **Blackline Tattoo Studio** é um site institucional moderno, totalmente responsivo e interativo projetado para um estúdio fictício de tatuagem especializado em arte autoral. A interface utiliza uma estética dark com alto contraste para transmitir a identidade forte do estúdio, aliando design imersivo a uma excelente experiência de navegação (UI/UX).

## 🚀 Funcionalidades Principais
* **Landing Page de Alto Impacto:** Hero section visualmente marcante com chamadas para ação (CTA) focadas em conversão.
* **Seção Sobre:** Espaço dedicado à história do estúdio, estilos (fine line, blackwork) e biossegurança.
* **Galeria de Artistas:** Apresentação da equipe de tatuadores com design limpo e organizado.
* **Carrossel de Depoimentos:** Slider interativo com feedback real de clientes para construir autoridade e confiança.
* **Seção de Contato e Localização:** Integração visual de mapa para facilitar a localização do estúdio no Rio de Janeiro.
* **Menu Mobile (Hambúrguer):** Navegação fluida e adaptada para dispositivos móveis com transições dinâmicas.

## 🛠️ Tecnologias Utilizadas
* **HTML:** Estruturação semântica do conteúdo.
* **CSS:** Estilização avançada com layouts modernos (Flexbox/CSS Grid), paleta de cores personalizada e responsividade (Media Queries).
* **JavaScript:** Responsável pela lógica de interatividade da página, incluindo a abertura/fechamento do menu mobile e o funcionamento dos sliders de depoimentos.

## 📱 Responsividade & Interatividade
O projeto foi desenvolvido sob o conceito de design responsivo, garantindo uma transição suave entre resoluções de desktop e smartphones. O uso de JavaScript garante que elementos de interface (como o menu de navegação em telas menores) funcionem de forma leve e nativa.
